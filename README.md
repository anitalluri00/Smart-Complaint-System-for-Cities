# Smart Civic Complaint System

An AI-powered platform for reporting and managing urban civic issues like potholes, garbage, and broken streetlights.

## Features

- 📸 **Image-based Complaint Reporting** with automatic classification using CNN
- 📍 **GPS Location Capture** for accurate issue mapping
- 🤖 **AI-Powered Analysis** with Groq LLM for summaries and resolutions
- 📊 **Advanced Analytics** including clustering, hypothesis testing, and Bayesian priority scoring
- 👥 **Role-Based Access** (Citizen/Admin)
- 📈 **Real-time Status Tracking**
- 🗺️ **Complaint Hotspot Detection**
- 📚 **RAG-based Query System** for past complaints

## Technology Stack

- **Frontend**: Streamlit
- **Backend**: Python
- **Database**: PostgreSQL
- **ML/DL**: TensorFlow, OpenCV, scikit-learn
- **AI**: Groq API (Mixtral-8x7b)
- **Deployment**: Docker, Docker Compose
